Compétences

Durant ma formation en tant que testeur logiciel QA, j'ai acquis des notions dans l'identification des défauts et anomalies afin de m'assurer que les applications logicielles respectent les normes de qualité de l'entreprise. J'ai appris à créer et exécuter des cas de tests, ainsi qu'à documenter les défauts de manière claire et précise. Grâce aux travaux dirigés (TD), j'ai développé des compétences en tests manuels, incluant la compréhension des langages HTML, CSS et JavaScript, essentiels pour tester les interfaces web.

En complément, j'ai exploré l'automatisation des tests avec Python et le framework Selenium, ainsi que l'outil Postman pour tester les API (GET, POST, PUT, DELETE). Ma formation m'a également permis de me familiariser avec Git pour gérer le versioning du code et collaborer efficacement avec les développeurs, notamment via le pair testing, afin de garantir une intégration fluide des modifications.
